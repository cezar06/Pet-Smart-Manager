2. Overall Description

2.1 Product Description

Our Pet Smart Manager web application has been developed for those that wish to visualize data and other things like a feeding calendar for their pets

2.2 Product Functions

The web application manages personal/family pets. These pets have associated data such as breed, age, name, owner, medical history, feeding calendar. The system offers support for visualizing and modifying this data. The admin is offered with support for managing users. The application offers the option of exporting different statistics in PDF format.

4. System Features

4.1 Home Page

Guests and users can read our welcome message and can see the pet of the week.

4.2 Login Page

Users of the app can login by entering their credentials and therefore access other functionalities of the app.

4.3 Register page

Guests can create an account and then use it to login into the application and access all the features the account existence provides.

4.4 About page

Users can view informations about the founders of the application and read a short description of our website.

4.5 Contact page

Users can send a feedback to the administrators of the website. The feedback consists in sending an image attachment or a message to the administrators.

4.6 Dashboard page

Users can add a pet to their account, edit the information about a certain pet, remove a pet, view all the pet information, and manage thier account.

4.7 Admin page

The admin can login into the app by entering his credentials and then be redirected to the admin page where he can manage users or see the feedback massages.

4.8 Terms and Conditions page

Users can read the terms and conditions of using the website by navigating to this page.
